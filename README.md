# 366日のコード

CodeZine「サンプルコード365+1」コンテスト応募作品（Haskell）

## 概要

翔泳社の創業を祝うプログラムです。会社名「SHOEISHA」をアスキーアートで表示し、創業日（1985年12月19日）からの周年情報を計算します。

## 機能

- 「SHOEISHA」のアスキーアート表示
- 創業からの年数と日数をリアルタイム計算
- 特別な日のお祝いメッセージ：
  - 🎉 創業記念日（12月19日）
  - ✨ キリ番（1000日ごと）
  - 🎊 周年（365日ごと）

## コード

完全な実装は `haskell/main.hs`（29行）にあります。

## 実行方法

```bash
cd haskell
runhaskell main.hs
```
